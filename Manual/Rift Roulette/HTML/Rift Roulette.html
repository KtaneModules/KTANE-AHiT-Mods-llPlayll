This module may not be immediately activated, and will display a grayscale crown or blob icon. If this happens, then there is an unsolved "Rhythm Maze" module on the bomb, solve all of them to activate the module.

Normally, this module will be in normal mode and will have 4 slots, refer to Section Normal Mode for operating the module in that case. However, if the bomb contains the module "Snatcher's Map", this module will enter Death Wish mode and will have 9 slots, refer to Section Death for operating the Death Wish mode.

Section Normal Mode
